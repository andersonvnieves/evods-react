# Getting Started

This guide will help you quickly set up and use EVODS components in your project.

## Installation

To install EVODS, use npm or yarn:

```bash
npm install evods-react
```

```bash
yarn install evods-react
```

## Prefix Convention

All EVODS React components use the `ev-` prefix for clarity and to avoid naming conflicts. For example:

- `ev-button`
- `ev-card`
- `ev-input`
